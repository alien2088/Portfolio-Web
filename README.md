# Portfolio-Web

## Overview

This is a personal portfolio website developed using HTML and CSS. It presents my background, projects, and contact information in a clean and responsive layout.

## Website Structure

The website is a single-page application divided into the following sections:

* **Header / Navigation Bar**
  Contains links to different sections (Home, About, Projects, Contact) for easy navigation.

* **Home Section**
  Displays profile image, name, and a short introduction.

* **About Section**
  Provides information about my background, education, and interests.

* **Projects Section**
  Showcases projects with descriptions and links to GitHub, demo videos, and documentation.

* **Contact Section**
  Includes contact details such as email, GitHub, and LinkedIn.

* **Footer**
  Displays basic information.

## Technologies Used

* HTML5
* CSS3

## Features

* Responsive design for mobile and desktop
* Smooth navigation
* Project showcase with external links
* Clean and structured user interface
